# walkiepaw-client

## 프로젝트 개발환경
- walkie-paw 팀의 프론트엔드 개발자들을 위한 개발환경 세팅 가이드입니다.
- 웹서버로 NGINX를 사용하고, 개발에는 React를 사용합니다.
- 백엔드 서버로의 요청은 `/api` 주소로 오고, 리버스 프록시 처리합니다.

## 생성 방법
- [Vite](https://ko.vitejs.dev/) 사용

## 주요 라이브러리 및 도구
- 라우팅 처리: React Router
- UI 컴포넌트 라이브러리: TailwindCSS
- 폼 처리: Formik, React Hook Form
- 상태 관리 라이브러리: Redux
